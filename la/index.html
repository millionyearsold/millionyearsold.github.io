Welcome to Lego Adelphi

Here you will find (hopefully) useful instructions and guidance for building the Lego Adelphi.

The Lego Adelphi is a small representation of the real building in London but made from about 980 pieces of lego instead of traditional construction materials. There are many benefits to using lego including:

  • Lego is relatively inexpensive
  • It's small and so easily transportable
  • Construction time is significantly reduced
  • There's no shortage of skilled labour
  • Lego buildings have a low eco footprint

Please note that there is a slight excess of building materials included with your package. This is to allow for even more artistic license and contingency (e.g. in the event that additional stability is required due to lack of structural integrity).

Step-by-Step Instructions

Step by step instructions provide the most detailed web based Lego Adelphi building guidance but there are some limitations. Most notably transparent building materials are invisible in the item list for each page. Another curiosity is that building process is not the same as was used during the design phase. It's possible that this could make some of the components more difficult to complete - particularly internal components in the latter stages of the build.

Visual Guide

The visual guide is a series of images showing the Lego Adelphi in various stages of design from start to finish. It is intended to complement the step-by-step instructions. Be aware that there was a system outage on the 8th/9th floor video screen representation during the design phase. Design Guide

The design guide utilizes the Lego Digital Designer application which is available for Mac and Windows. It is the best tool for understanding how to build Lego Adelphi but requires a PC of some sort. If using the Lego Digital Designer be sure to select "Building Guide Mode" (top right button - or F7). Note that like the step-by-step instructions the building process is not quite the same as that followed during the design phase and so that might make some components more challenging during the build.

Open the linked file below in Lego Digital Designer to access the design guide
